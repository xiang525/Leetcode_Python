# Leetcode_Python
* This is a repository for solutions to [Leetcode Online Judge Problems](https://oj.leetcode.com/). It helps you to prepare the interview of tech companies.

* Folder [Solutions](https://github.com/jinhw1989/Leetcode_Python/tree/master/Solutions) contains all the Python approaches to the problems, and [Hints](https://github.com/jinhw1989/Leetcode_Python/tree/master/Hints) contains explainations to the problems respectively. 

* All the problems are ordered by the created time, not by the difficulties of them. Some problems are still under consideration, and may not be the best approaches.
